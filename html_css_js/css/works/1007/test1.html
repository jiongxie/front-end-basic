Display 속성

Display속성은 요소를 어떻게 보여줄지를 결정한다.

속성의 처리 값 : visibility(보여주기), hidden(감추기)

none : 보여주지 않음

block : 블록 박스(영역차지)

inline : 인라인 박스(자신만의 공간확보)

inline-block : block과 inline의 합성
